hexo+github搭建个人博客

简单介绍

More info:hexo
网站可设置为中文版

hexo的github

准备工作

nodejs

-本地下载支持windows版的nodejs
（不一定需要）可以配置系统的环境变量，将nodejs加入进

github

-登陆github,如果没有账号请注册

-登陆网页创建仓库(repositories)，仓库名字必须为：.github.io

-本地下载git for windows客户端，可以选择其他git客户端
（不一定需要）可以配置系统的环境变量，将git加入进去

-配置SSH key
使用git-bash进行操作

如果提示：No such file or directory等字样，说明是第一次使用git

可以不填写内容直接回车三次，最终会在用户目录（c:\user\本机用户名）下生成.ssh\id_rsa.pub文件
用记事本打开并复制所有内容，打开github个人设置->SHH and GPG keys->New SSH key,将复制的内容粘贴到key的框内，title随意填写，最后保存设置

如果提示Are you sure you want to continue connecting(yes/no)?输入yes,回车
然后会提示成功配置，接着还要配置：

使用hexo

安装

在nodejs与git都搭建好的环境下，打开cmd

建站

完成安装后，进入一个合适的工作空间，执行下列命令

-_config.yml 网站的配置信息
-package.json 应用程序的信息
-scaffolds 模板文件夹，hexo的模板是指新建的markdown文件中默认填充的内容
-source 资源文件夹是存放用户资源的地方
-themes 主题文件夹

指令

编辑文章

执行命令hexo new 后，在source_posts\下会生成一个md文件，使用编辑器即可对该文件进行编辑，使用markdown的格式和hexo提供的标签编辑文章</p> <p>-默认生成内容有title,date,tags等<br>- <p class="article-more-link"> <a
